
This repository contains Jupyter Notebooks for exploring and working with NumPy.

## 🚀 Getting Started

1. Ensure you have Python installed.
2. Activate your virtual environment (if you are using one):
   ```bash
   # Windows
   .venv\Scripts\activate
   ```
3. Install the required dependencies (such as Jupyter and NumPy):
   ```bash
   pip install numpy jupyter
   ```
4. Start Jupyter Notebook or open the notebooks directly in VS Code:
   ```bash
   jupyter notebook
   ```
